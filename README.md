# 🎮 XO Game (Tic-Tac-Toe)

เกม XO แบบ GUI สร้างด้วย Python และ Tkinter
เล่นง่าย ดีไซน์สวย รองรับผู้เล่น 2 คน 💥

---

## 🌟 Features

* 🎯 ระบบสลับผู้เล่น X / O อัตโนมัติ
* 🏆 ตรวจสอบผู้ชนะครบทุกเงื่อนไข
* 🤝 รองรับผลเสมอ (Draw)
* 🔄 ปุ่ม Restart เล่นใหม่ได้ทันที
* 🌙 Dark Mode UI ดูสบายตา

---

## 🚀 วิธีใช้งาน

```bash
python xo_game.py
```

---

## 🎯 วิธีเล่น

* ผู้เล่น 2 คน ผลัดกันกด
* ใครเรียงครบ 3 ช่องก่อน ชนะ 🎉
* ถ้าเต็มกระดาน = เสมอ

---

## 🛠️ โครงสร้างโค้ด

* `check_winner()` → เช็คผู้ชนะ
* `on_click()` → จัดการการกด
* `reset_game()` → รีเซ็ตเกม

# 🧮 Modern Calculator (Tkinter)

เครื่องคิดเลข GUI ดีไซน์ทันสมัย
สร้างด้วย Python + Tkinter 💻✨

---

## 🌟 Features

* ➕➖✖️➗ คำนวณพื้นฐาน
* 🧹 ปุ่ม Clear ล้างค่า
* ⚡ คำนวณเร็วด้วย `eval()`
* 🎨 สีปุ่มแยกประเภทชัดเจน
* 🌙 Dark Mode UI

---

## 🚀 วิธีใช้งาน

```bash
python calculator.py
```

---

## 🎯 วิธีใช้

* กดตัวเลขเพื่อใส่ค่า
* กดเครื่องหมายเพื่อคำนวณ
* กด `=` เพื่อดูผลลัพธ์
* กด `C` เพื่อล้าง

---

## 🛠️ โครงสร้างโค้ด

* `press()` → เพิ่มค่า
* `clear()` → ล้างหน้าจอ
* `calculate()` → คำนวณผล

---

## ⚠️ หมายเหตุ

ใช้ `eval()`
เหมาะสำหรับการเรียนรู้ ไม่ควรใช้กับ input ที่ไม่ปลอดภัย
