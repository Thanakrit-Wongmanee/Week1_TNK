# 🎮 XO Game (Tic-Tac-Toe) ด้วย Python Tkinter

เกม XO (Tic-Tac-Toe) แบบ GUI ที่พัฒนาด้วยภาษา Python โดยใช้ Tkinter
ออกแบบเรียบง่าย เล่นได้ทันที รองรับผู้เล่น 2 คน 🎉

---

## 🧩 Features

✨ หน้าตา GUI สวยงาม โทน Dark Mode
✨ ระบบสลับผู้เล่น (X / O) อัตโนมัติ
✨ ตรวจสอบผู้ชนะครบทุกกรณี
✨ รองรับผลเสมอ (Draw)
✨ ปุ่ม Restart เล่นใหม่ได้ทันที
✨ ใช้งานง่าย ไม่ต้องติดตั้งอะไรเพิ่ม (ใช้ Tkinter มาตรฐาน)

---

## 🖼️ Preview

> หน้าจอเกมจะมีตาราง 3x3 พร้อมแสดงว่าเป็นตาของใคร

---

## 🚀 วิธีใช้งาน

1. ตรวจสอบว่ามี Python ติดตั้งแล้ว
2. รันไฟล์ `.py` นี้ได้เลย:

```bash
python xo_game.py
```

---

## 🎯 วิธีเล่น

* ผู้เล่น 2 คน ผลัดกันเล่น (X และ O)
* คลิกที่ช่องว่างเพื่อวางสัญลักษณ์
* ใครเรียงครบ 3 ช่องก่อน ชนะ 🏆
* ถ้าเต็มกระดาน = เสมอ 🤝

---

## 🛠️ โครงสร้างโค้ดหลัก

* `check_winner()` → ตรวจสอบผลแพ้ชนะ
* `on_click()` → จัดการเมื่อผู้เล่นกดปุ่ม
* `reset_game()` → รีเซ็ตเกมใหม่
* Tkinter UI → สร้างหน้าต่าง ปุ่ม และ layout

# 🧮 Modern Calculator (Python Tkinter)

เครื่องคิดเลข GUI ดีไซน์ทันสมัย สไตล์ Dark Mode
พัฒนาด้วย Python และ Tkinter ใช้งานง่าย รวดเร็ว 💻✨

---

## 🌟 Features

✨ หน้าตา Modern Dark UI
✨ รองรับการคำนวณพื้นฐาน (+ - × ÷)
✨ ปุ่ม Clear (C) ล้างค่าได้ทันที
✨ แสดงผลแบบ Real-time
✨ ป้องกัน Error เบื้องต้น
✨ ขนาดหน้าต่างคงที่ ใช้งานสะดวก

---

## 🖥️ Preview

> เครื่องคิดเลขแบบตาราง 4x4 พร้อมสีแยกประเภทปุ่มชัดเจน

---

## 🚀 วิธีใช้งาน

1. ตรวจสอบว่ามี Python ติดตั้งแล้ว
2. รันไฟล์:

```bash id="run1"
python calculator.py
```

---

## 🎯 วิธีใช้โปรแกรม

* กดตัวเลขเพื่อใส่ค่า
* กดเครื่องหมาย (+ - × ÷) เพื่อคำนวณ
* กด `=` เพื่อดูผลลัพธ์
* กด `C` เพื่อล้างหน้าจอ

---

## 🛠️ โครงสร้างโค้ดหลัก

* `press(num)` → เพิ่มค่าลงในหน้าจอ
* `clear()` → ล้างข้อมูลทั้งหมด
* `calculate()` → คำนวณผลลัพธ์ด้วย `eval()`
* Tkinter UI → สร้างหน้าต่างและปุ่มต่าง ๆ
